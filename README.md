# MessageEmailOut

在iOS越狱设备上写的一个毫无用处的小插件。 实现功能：如果手机不在身边而你又想查看未接来电、未读短信等消息，可以向手机发送一个命令启用新消息邮件转发功能，此后收到的消息便会转发到你指定邮箱。
